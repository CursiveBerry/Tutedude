# journalapp

A new Flutter project.

## Getting Started

This is a Journal taking app. The App uses tiles to show the content that user has given as input.
The user has to give input of a title and description and the system will automatically marks the date on the tile.

# The App have: 

- Homepage
- Content tile on the home_page with title,content,and date. 
- Add button to create new tile.
- Add button creates a bottom sheet.
- Bottom sheet have: 
    - text area for title.
    - text area for content.
    - Cancel button.
    - Submit button.
- Cancel button does not save the text and brings back to home page.
- Submit button creates a new tile with user input content at home page.
